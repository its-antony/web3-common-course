# Change: 完成 Part09 参与 Web3 的四种方式内容研发

## Why

Part09 是课程的关键章节之一，直接回答学员最关心的问题：「我应该如何参与 Web3？」。这一章节位于课程后半段，在学员已经建立了基础认知（Part1-2）、掌握了基本操作（Part3）、了解了赛道生态（Part4-5）、认识了风险陷阱（Part6-7）、知道了行业角色（Part8）之后，帮助学员根据自身情况选择合适的参与方式。

根据课程大纲 `Web3 通识课v2.md`，Part09 包含以下小节：
- 9.1 只学习、不入局的参与方式
- 9.2 小额参与的正确姿势
- 9.3 投资参与：从持币到交易
- 9.4 职业化参与的现实门槛

## What Changes

### 新增文档

1. **9.1-只学习不入局的参与方式.md**
   - 为什么「只学习」也是一种有效的参与方式
   - 学习型参与的价值：认知积累、机会储备
   - 如何进行有效的 Web3 学习
   - 学习资源推荐与学习路径规划

2. **9.2-小额参与的正确姿势.md**
   - 空投与撸毛生态简介
   - 空投的运作逻辑与常见模式
   - 小额交互的实操指南
   - 小额交互的风险控制
   - 成本收益分析与预期管理

3. **9.3-投资参与从持币到交易.md**
   - 现货持有 vs 合约交易的本质区别
   - CEX（中心化交易所）vs DEX 的选择
   - 仓位管理与投资心态
   - 定投、波段、长持的适用场景
   - 常见投资陷阱：FOMO、追涨杀跌、杠杆爆仓
   - **风险警示**：投资有风险，需谨慎决策

4. **9.4-职业化参与的现实门槛.md**
   - Web3 职业化参与的主要路径
   - 技术岗 vs 非技术岗的门槛差异
   - 从业者真实经验分享
   - 职业化参与的时间和机会成本
   - 进入 Web3 行业的实际建议
