## 转换规则

| Markdown 元素 | 幻灯片类型 | 说明 |
|--------------|-----------|------|
| 文件开头 `# 标题` | `section` | 每个 .md 文件的章节标题页 |
| `## 小节标题` | `content` | 内容页，标题 + 下方内容 |
| `> 引用文字` | `quote` | 引用页，大字居中 |
| `- 列表项` | `content` + `bullets` | 无序列表 |
| `1. 列表项` | `content` + `numbered` | 有序列表 |
| `| 表头 |` | `content` + `table` | 表格内容 |
| `:::info` 等 | `content` + `highlight` | 高亮提示框 |
| 文件末尾要点 | `summary` | 本节要点总结页 |

### 特殊页面

- **Part 标题页**: 每个 Part 开始时自动添加 `title` 类型
- **Part 结束页**: 每个 Part 结束时自动添加 `title` 类型

### 内容拆分原则

- 每页内容不超过 4-5 个要点
- 表格单独成页
- 长列表拆分为多页
