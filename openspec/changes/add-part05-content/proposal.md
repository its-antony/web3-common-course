# Change: 研发 Part05 Web3 主流赛道与商业模式速览

## Why

Part05 是 Web3 通识课的核心应用模块，帮助学员全面了解 Web3 各主流赛道的商业模式、价值来源和潜在风险。该部分内容涵盖 DeFi、NFT、GameFi、SocialFi、RWA、DAO、预测市场、Meme 币、AI+Crypto 等热门领域，是学员建立行业认知地图的关键章节。

**课程大纲要求的内容**：
- 5.1 DeFi：去中心化金融在解决什么问题
- 5.2 NFT 的价值来源与泡沫
- 5.3 GameFi：链游的经济模型与常见问题
- 5.4 SocialFi 与 DID（去中心化身份）
- 5.5 RWA：现实世界资产上链
- 5.6 DAO：去中心化自治组织
- 5.7 预测市场：Polymarket 与信息定价
- 5.8 Meme 币现象解读
- 5.9 AI + Crypto：新兴融合趋势
- 5.10 已被反复验证失败的常见模式

## What Changes

### 新增内容

**课程文档**（10 篇）：
1. **5.1-DeFi去中心化金融.md** - 讲解 DeFi 解决的问题、核心协议类型（AMM/DEX/借贷/收益聚合）
2. **5.2-NFT的价值来源与泡沫.md** - 解析 NFT 不只是"小图片"的多元价值（门票/身份/资产凭证）
3. **5.3-GameFi链游经济模型.md** - 分析链游的经济模型设计与常见问题
4. **5.4-SocialFi与DID.md** - 介绍去中心化身份与社交协议的核心概念
5. **5.5-RWA现实世界资产上链.md** - 讲解代币化、托管、预言机、储备证明（PoR）
6. **5.6-DAO去中心化自治组织.md** - 解析 DAO 的治理机制、投票逻辑与现实挑战
7. **5.7-预测市场Polymarket.md** - 介绍预测市场运作原理与 Polymarket 案例
8. **5.8-Meme币现象解读.md** - 分析 Meme 币的文化现象与投资风险
9. **5.9-AI与Crypto融合趋势.md** - 介绍 AI Agent、去中心化算力、数据市场
10. **5.10-已验证失败的模式.md** - 总结行业反复失败的常见模式与教训

### 配套资料
- 创建 `_research/` 目录存放研发笔记和资料清单
- 为每个小节准备必读和选读学习资料（S/A 级来源为主）
- 设计可验证的学习任务

### 学习任务设计
根据大纲要求，Part05 包含以下学习任务：
1. 做一张 TradFi vs DeFi 对照表，列出 3 个 DeFi 独特优势 + 1 个潜在风险
2. 用一句话解释 NFT 和比特币最大的不同
3. 用一句话解释 DID 和微信账号最大的不同
