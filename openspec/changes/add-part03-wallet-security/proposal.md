# Change: Part03 钱包、账户与资产安全课程内容研发

> **状态**: ✅ 已完成
> **完成日期**: 2025-01-09

## Why

Part 3 是 Web3 通识课的核心实操章节，帮助学员掌握 Web3 世界最基础也最重要的技能——安全地管理自己的数字资产。这是学员从"了解 Web3"到"参与 Web3"的关键一步。没有这个基础，学员无法安全地进行任何链上交互。

## What Changes

根据课程大纲 v2.0，Part03 包含以下 5 个小节：

- **3.1 钱包、地址、私钥、助记词的关系**
  - 核心概念讲解
  - 清晰的类比说明（银行账户 vs 钱包）
  - 关系图解

- **3.2 钱包与交易所的本质区别**
  - 托管 vs 自托管
  - 风险对比
  - 适用场景

- **3.3 授权、签名与交互的基本逻辑**
  - 什么是签名
  - 授权（Approve）机制解释
  - 常见交互类型

- **3.4 新手最容易踩的安全与资产管理坑**
  - 常见骗局类型
  - 安全最佳实践
  - 真实案例警示

- **3.5 实操：在测试网完成你的第一笔交易**
  - 创建钱包（MetaMask / OKX Wallet）
  - 添加测试网（BNB Chain Testnet / Sepolia）
  - 领取测试币
  - 完成链上转账
  - 在区块浏览器查看交易记录
  - 包含 Playwright 自动化截图指引
