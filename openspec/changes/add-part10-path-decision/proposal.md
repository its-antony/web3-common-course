# Change: 添加 Part10 最终路径建议与决策出口

## Why

Part10 是 Web3 通识课的关键章节，帮助学员在完成前面九个部分的学习后，做出明确的决策：是否继续深入 Web3 领域、选择哪条路径、以及如何正确止损抽身。这是课程的「决策出口」，确保学员能够根据自身情况做出理性选择。

## What Changes

根据课程大纲 v2.0，Part10 包含以下四节内容：

- **10.1 我是否适合长期投入 Web3**
  - 帮助学员进行自我评估，判断是否适合继续投入 Web3
  - 提供清晰的决策框架和评估维度

- **10.2 如果继续，最推荐的路径选择**
  - 针对不同类型的学员（学生、Web2 从业者等）提供路径建议
  - 结合 Part8/Part9 的内容，给出具体可执行的建议

- **10.3 如果不继续，如何止损与抽身**
  - 帮助决定不继续的学员安全退出
  - 提供资产处理、账户安全、心理调整等建议

- **10.4 是否进入下一阶段陪跑 / 专项课程**
  - 介绍后续学习选项
  - 帮助学员决定是否需要更深入的指导
