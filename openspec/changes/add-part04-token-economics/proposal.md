# Change: 研发 Part04 Token 与 Web3 的经济系统

## Why

Part04 是 Web3 通识课的核心经济模块，帮助学员理解 Token 的本质、稳定币的运作机制以及代币经济学的基础知识。这是学员从「理解技术」到「理解价值」的关键过渡，也是后续理解 DeFi、NFT 等应用的基础。

**课程大纲要求的内容**：
- 4.1 Token 的基本功能与作用
- 4.2 Token ≠ 股票 ≠ 积分：三者的本质区别
- 4.3 稳定币专题（USDT、USDC、DAI 的区别与风险；算法稳定币的失败案例）
- 4.4 通胀、解锁与激励的基础理解
- 4.5 为什么大多数 Token 最终会失败

**为什么重要**：
1. Token 是 Web3 经济系统的基石，理解其本质是参与 Web3 的前提
2. 稳定币是普通人最常接触的加密资产，理解其风险至关重要
3. 代币经济学知识帮助学员识别项目的可持续性，避免踩坑

## What Changes

### 新增内容

1. **4.1-Token的基本功能与作用.md** - 解释 Token 的定义、类型和核心功能
2. **4.2-Token股票积分的本质区别.md** - 对比三者的法律地位、权利属性和价值来源
3. **4.3-稳定币专题.md** - 深入讲解稳定币的机制、风险和 UST/Luna 崩盘案例
4. **4.4-通胀解锁与激励.md** - 介绍代币经济学的基础概念和大额解锁案例
5. **4.5-为什么大多数Token会失败.md** - 分析 Token 失败的常见原因和判断方法

### 配套资料

- 创建 `_research/` 目录存放研发笔记
- 为每个小节准备必读和选读学习资料
- 设计配套的学习任务
- 创建必要的 Mermaid 图表（稳定币机制、解锁时间线等）

### 重点案例

- **UST/Luna 崩盘**：算法稳定币失败的经典案例（4.3 节核心案例）
- **大额解锁崩盘**：某项目解锁导致价格崩盘的案例分析（4.4 节案例）
