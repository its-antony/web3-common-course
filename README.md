# Web3 通识课 v2.0

> **课程定位**：面向学生 / Web2 上班族的 Web3 通识课

## 课程目标

- 系统性建立完整 Web3 认知地图
- 搞清楚 Web3 是什么、钱从哪里来、风险在哪里
- 掌握基础实操技能，能独立完成链上交互
- 在课程最后给出明确的路径与决策建议

## 课程结构

| 模块 | 主题 | 内容概要 |
|-----|------|---------|
| Part 0 | 导论 | 课程使用说明与学习建议 |
| Part 1 | Web3 全局速览 | 快速建立 Web3 世界观 |
| Part 2 | 区块链与加密系统 | 技术底层的最小认知 |
| Part 3 | 钱包、账户与资产安全 | 必会的基础实操 |
| Part 4 | Token 与经济系统 | 理解 Web3 的经济逻辑 |
| Part 5 | 主流赛道与商业模式 | DeFi、NFT、DAO 等赛道速览 |
| Part 6 | 收益、风险与陷阱 | 识别风险与常见骗局 |
| Part 7 | 项目快速判断 | 建立项目筛选能力 |
| Part 8 | 行业角色与岗位 | Web3 职业全景 |
| Part 9 | 参与方式 | 四种参与 Web3 的方式 |
| Part 10 | 路径建议与决策 | 最终决策出口 |
| Part 11 | 资源导航 | 工具、平台与学习资料 |

## 适用人群

- 对 Web3 感兴趣但不知从何入手的学生
- 想了解 Web3 的 Web2 行业从业者
- 希望系统性学习而非碎片化了解的人

## 学习建议

1. 按顺序学习，每个 Part 都有前置依赖
2. 完成每个模块的学习任务，输出是最好的学习
3. 不急于「入场」，先建立完整认知
4. 善用课程提供的学习资料链接

---

> 本课程为「Web3 通识课」，旨在建立认知，不构成任何投资建议
