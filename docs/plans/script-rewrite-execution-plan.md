## 执行方式

### 方式一：串行执行（推荐）
在新对话中，每次处理 1-2 个 Part：
1. 指定要处理的 Part 编号
2. AI 读取对应的 PPT 和文档
3. 按照 `script-rewrite-rules.md` 撰写新讲稿
4. 输出到对应的 `partXX.md` 文件

### 方式二：并行执行
使用子代理并行处理多个 Part，但需注意：
- 每个子代理需要完整读取撰写规范
- 建议分批并行（如 Part 02-04 一批，Part 05-07 一批）

### 执行命令示例
```
请按照 docs/plans/script-rewrite-rules.md 的规范，重写 Part 02 的讲稿。

输入文件：
- PPT: slides/src/presentations/part02.ts
- 文档: Web3通识课课程内容/Part02-区块链与加密系统/*.md

输出文件：slides/src/scripts/part02.md
```
