# Change: 完成 Part06 收益、风险与结构性陷阱章节内容研发

## Why

Part 6 是课程中最关键的风险教育章节，帮助学员理解 Web3 世界中「钱从哪里来」以及「陷阱在哪里」。这是保护学员避免重大损失的核心内容，也是通识课区别于营销课的重要标志。

根据课程大纲 `Web3 通识课v2.md`，Part06 包含以下 7 个小节，均需完成内容研发：

- 6.1 Web3 收益从哪里来
- 6.2 高 APY 的真实含义
- 6.3 常见割人结构与庞氏模型（案例复盘：Luna/UST、FTX、各类土狗项目）
- 6.4 MEV 与隐性交易成本（什么是三明治攻击、抢跑交易）
- 6.5 跨链桥使用与风险
- 6.6 监管与合规风险（各国监管政策差异、税务问题基础认知）
- 6.7 普通人最容易处于的风险位置

### 各章节内容要点

#### 6.1 Web3 收益从哪里来
- 收益的四大来源分类：价格波动、质押奖励、流动性挖矿、空投激励
- 零和博弈 vs 正和博弈的区分
- 「收益来源不明 = 你可能就是收益来源」的核心警示
- Mermaid 流程图：不同收益来源的资金流向

#### 6.2 高 APY 的真实含义
- APY vs APR 的区别
- 高 APY 背后的代币通胀稀释
- 无常损失（Impermanent Loss）的基本概念
- 案例：某项目 10000% APY 的真实收益计算
- 表格：不同 APY 水平的风险提示

#### 6.3 常见割人结构与庞氏模型
- 庞氏模型的识别特征
- **案例复盘**：
  - Luna/UST：算法稳定币的死亡螺旋
  - FTX：中心化交易所的信任崩塌
  - 典型土狗项目特征
- 「拉盘-喊单-出货」的常见套路
- Mermaid 时序图：庞氏资金流转

#### 6.4 MEV 与隐性交易成本
- 什么是 MEV（最大可提取价值）
- 三明治攻击原理与防范
- 抢跑交易（Front-running）机制
- 普通用户的实际损失场景
- Mermaid 时序图：三明治攻击过程

#### 6.5 跨链桥使用与风险
- 跨链桥的工作原理
- 历史重大跨链桥安全事件（Ronin、Wormhole、Nomad）
- 使用跨链桥的注意事项
- 降低风险的实操建议

#### 6.6 监管与合规风险
- 主要国家/地区监管政策对比（美国、中国、新加坡、欧盟）
- 加密货币税务基础认知
- 合规风险对普通用户的影响
- 表格：各国监管态度对比

#### 6.7 普通人最容易处于的风险位置
- 典型风险场景汇总
- 「最后接盘者」的特征
- 建立个人风险边界的方法
- 课程总结与行动建议
